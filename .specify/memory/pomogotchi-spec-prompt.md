Develop Pomogothci, a pomodoro timer app. It helps users track focus and break sessions, as well as track how many times they drink water or take a break to move or stretch.
It should allow users to start and stop pomodoro sessions of 40 minutes focus and 10 minutes break. There will be no login for this application as this is just the very first testing thing to ensure that our basic features are set up. This initial version will not support custom session lengths or break types. The main display of Pomogotchi is a simple timer with a large display showing the remaining time and a large start/stop button. A user starts a focus session by clicking the start button, and the timer will begin counting down. The are allowed to pause the session or stop early. When the session is over, a visual indicator shows the user that the session is complete and a call to action prompts the user to take a 10 minute break. A user logs hydration events with one click, and if no events have been logged in a pre-defined period, an indicator and reminder is shown to drink water.
